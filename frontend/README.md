
# Smart Meeting Assistant - Frontend

This is the React 18, Tailwind CSS, and Radix UI powered frontend interface for the Smart Meeting Assistant. Everything is bundled via Vite.

## Running the code

Make sure you have Node 18+ installed.

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```
   *Note: Standard HTTP protocol will block the Web Audio API unless the app is accessed on `localhost`. For mobile/LAN testing, Vite will serve the project locally over a self-signed HTTPS certificate.*

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Production Preview:**
   ```bash
   npm run preview
   ```
